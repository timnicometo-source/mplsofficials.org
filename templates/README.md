# MOA Page Templates

Copy one of these files when creating a new page:

- `page-standard.html` — normal interior page with a hero.
- `page-featured.html` — larger hero and action buttons.
- `page-utility.html` — no hero; useful for forms, payment embeds, or focused pages.
- `page-hub.html` — card-based hub page.

After copying a template, update the title, meta description, `body data-page`, stylesheet/script paths, and page content.
