# Minneapolis Officials Association Static Site

Starter static site for moving MOA from Wix to VS Code, GitHub, and Hostinger.

## Main workflow

1. Edit files locally in VS Code.
2. Preview with a local server.
3. Commit changes to GitHub.
4. Deploy through Hostinger Git Deploy.

## Placeholder items

Many links/forms/images are intentionally set to `#` so final Square links, Google Forms, Google Calendar embeds, PDFs, and images can be added later.
